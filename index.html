CPU de référence

    Comment trouver mon CPU ?
    • Windows : Ctrl+Maj+Échap → onglet Performance → Processeur (nom en haut à droite).
    • Ou Win+R → dxdiag → ligne « Processeur ».
    • macOS : → À propos de ce Mac.

    Le score PassMark sert de référence théorique pour repérer un PC bridé (EDR, VPN, throttling…).

    Pour mesurer l'impact d'un EDR ou d'un VPN, lancez un test avec puis sans, et comparez les deux runs.
